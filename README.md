# pbhlt-7101-project-1

Project 1 repo for PBHLT 7101 - Data Management, Analytics and Presentation for Health Research at the University of Utah.

## Getting Started

Steps:

1. Download data (ansur2allV2.dta) from class page
2. Clone this project into the folder of your choosing
3. Move the downloaded data into the [data/](./data/) folder 
4. Using stata, open [SwigerProject1.do](./SwigerProject1.do)
5. Update the working directory file path to point to the directory you copied the project into
6. Run SwigerProject1.do
